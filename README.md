# What is Ara?
Ara is a advanced AI and ML system that can run on everthing from iot devices to massive servers. it is also free and open source under the GNU General Public License (see the file LICENSE for more info).
# what is ara scripts
The ara scripts ara the scripts that make ara work these scripts will be used in to a Docker container for scalable use.
# What do the scripts do?
* Ara nstart is the main script, it can learn from the internet, sensors, or manually adding databases
* Ara speach main is the tool to create a ara voice model
* Ara vision main is the tool to create a ara vision DB
